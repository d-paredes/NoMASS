670 ## Compiling on Linux {#Compiling-on-Linux}
671 ## Compiling on Linux
672 
673 The compilation process has to be completed after modification in the source code has been done.
674 These instructions are written around Ubuntu 14.04 LTS, but they might remain valid for most of Linux distributions.
675 
676 1. Install the g++ compiler with Fortran utilities to compile the code.
677 Fortran utilities are used to process EnergyPlus output files.
678 The g++ compiler can be installed from terminal with the following command.
679 ```sh
680 sudo apt-get install g++ gfortran
681 ```
682 
683 2. Install CMake tools. A terminal-based user insterface can be installed with the following command.
684 ```sh
685 sudo apt-get install cmake cmake-curses-gui
686 ```
687 Version of installed programmes can be verified with the following commands from a terminal.
688 ```sh
689 g++ --version
690 cmake --version
691 ```
692  \htmlonly <style>div.image img[src="gcompiler_cmake_version.png"]{width:600px;}</style> \endhtmlonly
693  @image html gcompiler_cmake_version.png "g++ compiler and cmake installed versions"
694 @image latex gcompiler_cmake_version.png "g++ compiler and cmake installed versions" width=10cm
695 3. Navigate in the terminal to the root of the EnergyPlus source code folder and create the build folder where the code will be compiled.
696 ```sh
697 cd /home/daps/Documents/No-MASS-GUI/EnergyPlus-8.6.0
698 mkdir build
699 cd build
700 ```
701 
702 4. Launch CMake from the build location by executing the following command from terminal.
703 ```sh
704 ccmake ../
705 ```
706  where `../` referes to the build's parent folder (root folder).
707 
708 5. Configure the build by pressing ***c***, and a list of editable build options will be presented.
709 Set the CMAKE_BUILD_TYPE type to "Release" and turn on the BUILD_FORTRAN options at this step.
710 Press ***c*** again to reconfigure after completing changes, then ***g*** to generate makefiles and exit.
711 @image latex ccmake_gui_step2.png "Build options" width=10cm
712  \htmlonly <style>div.image img[src="ccmake_gui_step2.png"]{width:600px;}</style> \endhtmlonly
713  @image html ccmake_gui_step2.png "Build options"
714 6. Finally, run `make -j N`, where `N` is the number of job slots to execute multiple jobs at once.
715 The default number of job slots is one, which means one job at time.
716 The number of job slots depends on the available hardware resources (CPU cores and memory), and the memory required by each make job.
717 @image latex make_output.png "`make -j 2` output" width=10cm
718  \htmlonly <style>div.image img[src="make_output.png"]{width:600px;}</style> \endhtmlonly
719  @image html make_output.png "`make -j 2` output"
720 7. The compiled EnergyPlus can be found in the `build/Products` folder.
721 @image latex EPlus_exeFolder.png "EnergyPlus and ReadVarsESO applications folder" width=15cm
722  \htmlonly <style>div.image img[src="EPlus_exeFolder.png"]{width:800px;}</style> \endhtmlonly
