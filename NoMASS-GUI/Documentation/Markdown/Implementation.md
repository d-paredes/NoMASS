<div id='Introduction'/>

This chapter describes the use of the No-MASS-GUI to launch building performance simulations with integration of the No-MASS platform.
No-MASS-GUI is coded in Python.
Therefore, a Python interpreter is required to launch the application.
There are multiple Python interpreters for both Llinux and Windows operating systems.

Anaconda2 is an open source platform that integrates Python and data science and machine learning libraries.
(for Python 2.7)
